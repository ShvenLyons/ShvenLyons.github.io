Research

TINT, works toward building a Trustworthy Internet of Intelligence with a tint of integrity, with research directions spanning Secure & Reliable Internet Infrastructure, Trustworthy AI Systems, Secure Data Analytics, Management & Governance, and Trustworthy Agent Interconnection.

My work follows this scope by examining privacy, security, and malicious-use risks in AI systems, especially RAG, agentic workflows, and tool-augmented applications.

Direction 01

Privacy-Aware AI Systems

Privacy-aware AI systems study how intelligent applications can reduce unnecessary data exposure while still supporting useful, context-sensitive behavior. Core topics include data minimization, consent boundaries, secure context handling, privacy-preserving inference, and responsible lifecycle management for user data.

Direction 02

Trustworthy Retrieval-Augmented Generation

RAG combines external knowledge retrieval with language generation to improve grounding and factuality. Trustworthy RAG focuses on retrieval quality, evidence faithfulness, source attribution, context governance, leakage resistance, and evaluation methods that make generated answers easier to verify.

Direction 03

Secure Agent Network

Secure agent networks examine how multiple AI agents coordinate across tools, memory, retrieval modules, and user-facing tasks. Unlike MAS, this direction focuses on cooperation among agents across broader domains and arbitrary tasks. Core research questions include agent identification, agent differentiation, and agent discovery, together with permission control, communication boundaries, failure containment, and auditability.

Current Focus

  • A current thread is the formalization of context intervention under modification functions.
  • Another direction explores how agent value can be distinguished in the design of scalable agent networks.
  • I am also using broadly disclosed interfaces to enhance RAG robustness and examine the effectiveness of existing PETs.
