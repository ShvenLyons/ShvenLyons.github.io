Our Group centers on <span class="tint-full"><span class="tint-initial">T</span>rustworthy, <span class="tint-initial">I</span>nterpretable, and <span class="tint-initial">N</span>ext-generation AI <span class="tint-initial">T</span>echnologies</span> (<a class="tint-link" href="https://tint-research.com/" target="_blank" rel="noopener">TINT</a>).

My work follows this scope by examining privacy, security, and malicious-use risks in AI systems, especially RAG, agentic workflows, and tool-augmented applications.

<section class="research-overview" aria-label="Research directions">
  <article class="research-card">
    <p class="research-card__label">Direction 01</p>
    <h2>Privacy-Aware AI Systems</h2>
    <p>
      Privacy-aware AI systems study how intelligent applications can reduce unnecessary data exposure while still supporting useful,
      context-sensitive behavior. Core topics include data minimization, consent boundaries, secure context handling, privacy-preserving
      inference, and responsible lifecycle management for user data.
    </p>
  </article>

  <article class="research-card">
    <p class="research-card__label"><a href="{{ '/research/projects/BenchRAG.html' | relative_url }}">Direction 02</a></p>
    <h2>Trustworthy Retrieval-Augmented Generation</h2>
    <p>
      RAG combines external knowledge retrieval with language generation to improve grounding and factuality.
      Trustworthy RAG focuses on retrieval quality, evidence faithfulness, source attribution, context governance, leakage resistance,
      and evaluation methods that make generated answers easier to verify.
    </p>
  </article>

  <article class="research-card">
    <p class="research-card__label">Direction 03</p>
    <h2>Secure Agent Network</h2>
    <p>
      Secure agent networks examine how multiple AI agents coordinate across tools, memory, retrieval modules, and user-facing tasks.
      Unlike MAS, this direction focuses on cooperation among agents across broader domains and arbitrary tasks.
      Core research questions include agent identification, agent differentiation, and agent discovery, together with permission control,
      communication boundaries, failure containment, and auditability.
    </p>
  </article>
</section>

## Current Focus

- A current thread is the formalization of context intervention under modification functions.
- Another direction explores how agent value can be distinguished in the design of scalable agent networks.
- I am also using broadly disclosed interfaces to enhance RAG robustness and examine the effectiveness of existing PETs.
